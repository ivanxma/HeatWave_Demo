Demo

What this demo covers

This demo walks through the main HeatWave capabilities in one application: natural-language SQL, AutoML operations, Lakehouse external tables, and GenAI vector-search workflows against your connected MySQL environment.

HeatWave NL_SQL

Ask business questions in natural language, review the generated SQL, inspect the referenced tables, and compare the SQL with the returned result sets.

HeatWave ML

Initialize demo datasets such as Iris, run HeatWave AutoML operations like ML_TRAIN, and monitor model-related activity and progress from the UI.

External Table with Lakehouse

Browse Lakehouse databases and tables, inspect table definitions, manage refresh behavior, and work with external data without loading it into native InnoDB tables first.

Performance with airportdb

Review HeatWave performance views and sample query behavior using the airportdb demo schema when it is available on the connected server.

GenAI / VectorStore

Create a knowledge base from URL content, embed and store vectors with HeatWave, then search the vector table and generate answers from the matched documents.
